### 分布式系统

分布式系统算法可以根据不同的性质进行分类。比如说根据时序模型，根据进程间的通信方式，根据这个算法所使用的故障模式，还有很多很多，我们接下来也会看到。

* 时序模型 (Timing Model)
* 进程间通信 (Interprocess Communication)
* 故障模式 (Failure Modes)
* 故障检测器 (Failure Detectors)
* 领袖选举 (Leader Election)
* 一致性 (Consensus)
* 仲裁集 (Quorums)
* 分布式系统中的时间 (Time in distributed systems)
* FLP 浏览
* 总结
* 引用
